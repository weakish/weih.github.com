What have I done

Geekdoor ( www.geek-door.com )

Wowarmory Recorder ( www.wowarmory.me )
魔兽世界英雄榜记录器，如果你曾经山口山过，那看名字你就应该能知道这是做什么的。它会记录你所登记的角色每一天的英雄榜，以便记录下角色的成长历史。在如此快餐的年代，可能对绝大多数玩家的意义不大，不过我想还是会有一些与我有相同想法的玩家，一直希望官方能提供类似的功能，不过却迟迟看不到影子。现在这也是完成了我在不懂Web时的一个小小的心愿，我也希望这里可以帮助到有同样需求的玩家们。实现方法没什么特别的，代码十分简单，你可以在Github上找到源代码。（应用跑在Linode，同前者，也可能需要挂代理）
TCP/IP Protocol Stack
看到这里你可别想多了，虽然各个操作系统都有实现自己的TCP/IP协议栈，但这个以C++编写的协议栈仅仅实现了最简单的IP、TCP层以及HTTP层协议的分析，这是去年下半年我在公司主要负责的事情。
